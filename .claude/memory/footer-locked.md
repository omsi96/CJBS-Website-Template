# The footer is owner-maintained

`src/components/site/Footer.tsx` is shared by **every** project in the workshop cohort, so
changes to it ripple across all sites. Treat it as deliberate, not casual.

**Rules:**
- Don't edit, restyle, or replace the footer on a whim — it's a shared, cohort-wide component.
- Don't remove it from `src/app/layout.tsx` (it renders on every page).
- Footer changes are a facilitator / repo-owner decision. If a student requirement seems to need
  one, flag it. If the facilitator or repo owner asks directly, go ahead.

Everything *above* the footer is yours to build.
